# Assignment Tracker

A very simple school assignment tracker built with HTML, CSS, and JavaScript.

## What it does
- Lets you choose a week from a dropdown
- Shows all 5 classes for that week
- Lets you add assignments
- Lets you check assignments off when finished
- Saves your progress in your browser on that device

## Files
- `index.html` = the page structure
- `style.css` = the design and mobile layout
- `script.js` = the app behavior and saved data

## Important note
Your data is saved in your browser on the device you use. If you clear browser storage or use a different device, the saved assignments will not move over automatically.
